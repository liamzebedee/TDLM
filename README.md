TDLM
====

![Desktop](https://i.imgur.com/uGRz2o8.png)

Beautiful collab Spotify playlists, written using Meteor, ES6 and Bootstrap 4.

 - Responsive
 - 👔 Transitions, 🌈 colour and everything nice
 - 👍 Upvoting / ranking
 - Soundcloud-style waveform in the mix
 - 🔒 Secure
 - 🛀 Clean JS code
